# neosite
This is a fanmade website I made to demonstrate and consolidate my knowledge of HTML, CSS and JavaScript – using content related to NCT.

This website features:
1. Album Rankings – my personal rankings of different units albums and descriptions of why I chose it.
2. An Interactive Timeline – showing key changes in NCT's lineups.
3. Content Recommendations – including a random content generator, this gives fans a new way to access material, also adopted with personal ratings.
4. Quizzes – interactive quizzes for fans.

I also have plans for future development to hopefully add minigames, additional quizzes, improved UI and animations and more.
